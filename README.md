# Student Management System

A responsive **Student Management System** built using **Node.js, Express.js, EJS, HTML, CSS, and JavaScript** that demonstrates complete **CRUD (Create, Read, Update, Delete)** operations. The application allows users to manage student records through an intuitive dashboard where they can add new students, view detailed information, edit existing records, and delete students. Each student is assigned a unique ID using UUID, and the project follows RESTful routing with Express and Method Override for handling `PATCH` and `DELETE` requests.

## Features

* Add new students
* View all students in a dashboard
* View individual student details
* Edit student information
* Delete student records
* Unique Student ID generation using UUID
* Responsive user interface for desktop and mobile devices
* RESTful CRUD architecture
* Clean and organized project structure

## Tech Stack

* Node.js
* Express.js
* EJS
* HTML5
* CSS3
* JavaScript
* UUID
* Method Override

## Learning Outcomes

This project helped in understanding Express routing, EJS templating, CRUD operations, form handling, RESTful APIs, method overriding, and building responsive web applications using Node.js.
